|           | ArrayList       | LinkedList |
|-----------|-----------------|------------|
| 存储方式      | 连续内存空间          | 分散内存空间     
| 容量扩展      | 长度不可变           | 可灵活扩展      |
| 内存效率      | 元素占用内存少、但可能浪费空间 | 元素占用内存多    
| 访问元素      | O(1)            | O(n)       |
| 添加元素到任意位置 |O(n)             | O(1)       |
| 删除元素      |O(n)             | O(1)       |
